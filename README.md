# Futsalock

Futsalock is an innovative platform designed for the seamless booking and management of futsal grounds. It allows users to register, manage grounds, and book slots effortlessly online. This application is built using .NET for backend services and Angular for the frontend user interface.

## Features

### User Authentication
- Register a new account or login to access personalized features.
- Securely log out of the application.

### Booking Management
- Make new bookings for available futsal grounds.
- Admins can manage booking statuses and payment confirmations.

### Ground Management
- View available grounds for booking.
- Ground owners can add new grounds and update existing ground details.
- Owners can manage and view all grounds under their management.

### User Profile
- Access and manage user profile information.
